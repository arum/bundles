Glue

Glue is an OSGi bundle that provides an mechanism for OSGi service based dependency injection.

To declare a component add the following manifest header to your bundle referencing each class you want to be created and injected with OSGi services:

Glueable-Component: fully.qualified.classname[(, fully.qualified.classname)*]

As a minimum each component must implement {@link Glueable} and a default constructor.

The activation lifecycle for a glueable component is as follows:

  • Bind all required/optional services. See {@link Glueable}
  • Bind the bundle context if required. See {@link Contextual}
  • Activate. See {@link Activatable}
  • Register services. See {@link GlueableService}

The deactivation lifecycle for a glueable component is essentially the reverse of the above, as follows:

  • Unregister services.
  • Deactivate.
  • Unbind the context.
  • Unbind required/optional services.
